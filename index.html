Portabilité vers Jawhara Mobile

La portabilité vous permet de garder votre numéro de téléphone tout en changeant d’opérateur. Avec Jawhara Mobile, profitez d’un transfert simple et rapide sans perdre votre numéro actuel. Connecté avec Jawhara Mobile !

Formulaire de portabilité

Comment transférer votre numéro vers Jawhara Mobile :

  1. Rendez-vous au centre TT le plus proche : Un conseiller vous aidera à initier le transfert de votre numéro depuis votre opérateur actuel.
  2. Obtenez et présentez votre code RIO :
    • Composez gratuitement *172# pour recevoir un message contenant votre identité, votre période d’engagement, et votre code RIO (Relevé d’Identité Opérateur).
    • Remplissez le formulaire de portabilité en choisissant l’offre Jawhara Mobile qui vous convient.
  3. Demandez votre carte SIM Jawhara Mobile : Celle-ci sera activée sous 6 jours à partir de la demande, et pendant ce délai, votre numéro actuel restera actif.
  4. Recevez une confirmation par SMS : Un message de confirmation vous sera envoyé sur votre ancien numéro avec les détails du transfert (date et heure). Une fois ce transfert finalisé, vous pourrez utiliser votre ligne Jawhara Mobile comme auparavant.

Et laissez Tunisie Telecom s’occuper de tout le reste !

Important à savoir :

  • Le transfert est totalement gratuit.
  • Il n’entraîne aucune coupure de service pendant le processus.
  • Il est possible de conserver votre numéro même si vous avez des créances impayées avec votre opérateur actuel.
